tableView 分割线

当系统适配到iOS7的时候,cell的分隔线短了15像素,这是因为在iOS7中,UITableViewCell左侧会有默认的15像素的空白,这时候,设置setSeparatorInsets:UIEdgeInsetsZero 就能将空白去掉.但是在iOS8中,设置setSeparatorInsets:UIEdgeInsetsZero就不起作用了,解决方法如下:在viewDidLoad 中加入以下代码

`if ([self.discussTableView respondsToSelector:@selector(setSeparatorInset:)]) {

    [self.discussTableView setSeparatorInset:UIEdgeInsetsZero];

}

if ([self.discussTableView respondsToSelector:@selector(setLayoutMargins:)]) {

    [self.discussTableView setLayoutMargins:UIEdgeInsetsZero];

}
`

在tableView的代理方法中加入以下代码

`- (void)tableView:(UITableView )tableView willDisplayCell:(UITableViewCell )cell forRowAtIndexPath:(NSIndexPath *)indexPath

{

if ([cell respondsToSelector:@selector(setSeparatorInset:)]) {

[cell setSeparatorInset:UIEdgeInsetsZero];

}

if ([cell respondsToSelector:@selector(setLayoutMargins:)]) {

[cell setLayoutMargins:UIEdgeInsetsZero];

}

}`
